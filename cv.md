## **Сумма чисел Фибоначчи до 1кк**

```python 
def fib_gen():
    c=1
    p=0
    while(True):
        yield c
        c,p=c+p,c
 
 
for f in fib_gen():
    print(f)
    if (f > 1000000):
        break
```
